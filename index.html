Detection of DGA Domains by Exploring Different Machine Learning Approaches

Developed and evaluated various machine learning models to detect malicious website domains generated by Domain Generation Algorithms (DGAs) using Keras.

Processed over 200,000 domains from built-in logs and external databases, leveraging Scikit-Learn for feature extraction. Designed and deployed an interactive web application using Streamlit, making the best-performing model accessible for industrial applications.

3D Point Cloud Table Detection and Segmentation

Designed and evaluated multiple pipelines for detecting and segmenting tables in indoor environments using 3D point cloud data. Built a depth-based classification system using DGCNN on real and synthetic datasets, and implemented a monocular depth estimation approach (ZoeDepth) to extend functionality to RGB-only inputs.

Developed both binary classification and per-point segmentation models, tested across SUN3D and RealSense datasets. Employed evaluation metrics including accuracy, F1-score, and depth error analysis. Visualized predictions and model performance to assess generalization to real-world scenes.

EKF-MPC Robot Navigation

Developed a unified navigation system for autonomous wheeled robots by integrating Extended Kalman Filter (EKF) for localization and Model Predictive Control (MPC) for trajectory tracking.

Implemented EKF to estimate the robot’s pose using range and bearing measurements from known landmarks, and employed MPC to optimize control inputs. Evaluated system performance through simulation (PyBullet), comparing trajectories with and without EKF integration, and demonstrated improved accuracy, stability, and noise resilience.

Tic-Tac-Toe Shape Detection and Pick & Place Manipulation

Designed and implemented a multi-stage pipeline for a Franka Emika Panda robot arm to autonomously detect, classify, and manipulate Tic-Tac-Toe-like shapes. The project was structured around three tasks: accurate pick-and-place of shapes using grasping strategies, robust classification on variable-height tables, and identifying and manipulating the most frequent shape.

Integrated ROS with PCL for point cloud processing, OpenCV for shape recognition, and MoveIt for motion planning. The solution included dynamic height detection, orientation estimation, and error recovery mechanisms.
